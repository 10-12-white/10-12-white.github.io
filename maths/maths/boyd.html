Convex Optimization Notes: Fundamental Sets

Convex Set

Intuition: The "Line of Sight" rule. If you are at any point in the set, you can see any other point in the set without your path leaving the boundary.

Formal Definition: A set $C$ is convex if for every $x, y \in C$ and $\theta$ such that $0 \leq \theta \leq 1$, we have:

$\theta x + (1 - \theta)y \in C$

This creates a line segment between two points.

Affine Set

Intuition: A set that contains the entire infinite line passing through any two of its points (not just the segment). An affine set can be thought of as the largest extension of a linear relationship.

Formal Definition: A set $C$ is affine if for every $x, y \in C$ and $\theta \in \mathbb{R}$:

$\theta x + (1 - \theta)y \in C$

Note: Every affine set is also a convex set, but not every convex set is affine.

Closure ($\mathbf{cl } S$): The union of the set and its boundary.

A set is closed if it contains all its limit points. In optimization, we usually deal with closed sets to ensure that we can actually "reach" the optimal point on the boundary.

$\mathbf{cl } S = \{x \mid \forall \epsilon > 0, \{y \mid \|y-x\| \leq \epsilon\} \cap S \neq \emptyset\}$

Bounded vs. Unbounded Sets:

A set is bounded if it can be contained within a ball of finite radius $R$.

A set is unbounded if it extends to infinity in some direction. For example, a ray or a halfspace is an unbounded convex set.

Key Concept: If a convex optimization problem has an unbounded feasible set, the optimal value might be $-\infty$.

Interior Sets

The interior of a set is the "inner region" of a set. Put more clearly, a point is in the interior if you can move a tiny bit in any direction and still stay inside the set. You consider that this point can be enclosed in a "ball" that will fit perfectly inside this set.

Formal Definition: $x \in int(S)$ if there exists an $\epsilon > 0$ such that a ball of radius $\epsilon$ around x is entirely contained in $S$. You aren't "touching the fence"; you have a buffer zone.

Hyperplane

Intuition: A flat surface that divides the space into two halves. In 3D it's a plane; in 2D it's a line. It is used to describe the shape of a space.

Formal Definition: A set of the form:

$\{x \mid a^T x = b\}$

Where $a$ is a non-zero vector (the normal vector) and $b$ is a constant.

Supporting Hyperplane: A special case where the hyperplane touches the "boundary" of a convex set without cutting through its interior.

Affine Hull

Intuition: The smallest flat "space" (line, plane, etc.) that can contain a given set of points.

Formal Definition: The set of all affine combinations of points in $S$:

$\mathbf{aff } S = \{ \theta_1 x_1 + \dots + \theta_k x_k \mid x_i \in S, \sum \theta_i = 1 \}$

Convex Hull

The convex hull of a set $S$, denoted $\text{conv } S$, is the set of all convex combinations of points in $S$.

$$\text{conv } S = \{ \sum \theta_i x_i \mid x_i \in S, \theta_i \ge 0, \sum \theta_i = 1 \}$$

Halfspaces

From earlier, we know that a hyperplane is the set $\{x \mid a^T x = b\}$.

A halfspace is the set $\{x \mid a^T x \le b\}$.

Hyperplanes are the boundaries; halfspaces are the regions on one side of those boundaries.

Sets and Compliments

From earlier, we know that a set is a collection of points, which we can call $C$..

The compliment is the set points not contained in $C$. \{x \mid a^T x \le b\}$.

The PSD Cone ($\mathbb{S}^n_+$): The set of symmetric $n \times n$ matrices that are positive semidefinite.

A matrix $X$ is in the PSD cone if:

$z^T X z \ge 0 \quad \forall z \in \mathbb{R}^n$

Equivalently, all eigenvalues of $X$ are non-negative ($\lambda_i \ge 0$).

Significance: This cone is the foundation of Semidefinite Programming (SDP), one of the most powerful tools in modern optimization.

Positive Definite ($\mathbb{S}^n_{++}$): The "Interior" of the PSD cone.

Matrices where $z^T X z > 0$ for all $z \neq 0$. All eigenvalues are strictly positive.

Convex Optimization Notes: Fundamental Sets

Convex Set

Intuition: The "Line of Sight" rule. If you are at any point in the set, you can see any other point in the set without your path leaving the boundary.

Formal Definition: A set $C$ is convex if for every $x, y \in C$ and $\theta$ such that $0 \leq \theta \leq 1$, we have:

$\theta x + (1 - \theta)y \in C$

This creates a line segment between two points.

Affine Set

Intuition: A set that contains the entire infinite line passing through any two of its points (not just the segment). An affine set can be thought of as the largest extension of a linear relationship.

Formal Definition: A set $C$ is affine if for every $x, y \in C$ and $\theta \in \mathbb{R}$:

$\theta x + (1 - \theta)y \in C$

Note: Every affine set is also a convex set, but not every convex set is affine.

Hyperplane

Intuition: A flat surface that divides the space into two halves. In 3D it's a plane; in 2D it's a line. It is used to describe the shape of a space.

Formal Definition: A set of the form:

$\{x \mid a^T x = b\}$

Where $a$ is a non-zero vector (the normal vector) and $b$ is a constant.

Supporting Hyperplane: A special case where the hyperplane touches the "boundary" of a convex set without cutting through its interior.

Affine Hull

Intuition: The smallest flat "space" (line, plane, etc.) that can contain a given set of points.

Formal Definition: The set of all affine combinations of points in $S$:

$\mathbf{aff } S = \{ \theta_1 x_1 + \dots + \theta_k x_k \mid x_i \in S, \sum \theta_i = 1 \}$
